# Voice Recordings Directory

Place WhatsApp voice recordings here.

## Export from WhatsApp
1. Open the voice message in WhatsApp
2. Tap and hold > Share/Forward > Save to Files
3. Rename and place here

## Naming Convention
- Use lowercase with hyphens: `mom-dad.mp3`, `sarah.mp3`
- Match the filename in `data/dedications.json`

## Supported Formats
- MP3 (recommended)
- WAV
- OGG
- M4A
